Windows 강제 키 치환 레지스트리 코드 생성기

by. 현록玄鹿_Blackdeer

실제 키보드의 키를, → 키로 사용하고 싶습니다.

    대상:        

원키와 변환키가 같은 규칙은 자동으로 제거합니다. (빨갛게 표시됨)

새로고침(F5)를 누르면, 설정한 모든 규칙이 삭제되고 초기화됩니다.

사용법

 

[Win]+[R]로 실행을 열어, notepad를 입력하여 메모장을 엽니다.

(직접 열어도 됩니다. 아무 텍스트에디터나 가능합니다.)

 

복사한 코드를 붙여넣고, .reg 파일로 저장합니다.

 

저장된 파일을 실행합니다.

 

재부팅이나 로그오프/로그온 하여 확인합니다.
